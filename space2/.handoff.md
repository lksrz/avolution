## Architecture
- `predictor.py` - Main engine. Run `python3 predictor.py`
  - Fetches 20 candles from Binance
  - Verifies previous prediction if target candle closed
  - Updates predictions.log with actual results (or updates prediction if changed)
  - Makes new prediction using 10-signal model
  - Saves state.json
- `state.json` - Current window state, tracks what we predicted
- `predictions.log` - Append-only log (updates PENDING entries in-place)

## Key Principles (from MEMORY.md)

- Pure market signals only — no self-referencing meta-signals
- Simple > complex — each signal must be independent
